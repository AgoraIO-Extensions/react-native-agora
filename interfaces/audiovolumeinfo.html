Interface AudioVolumeInfo

Properties of the audio volume information. An array containing the user ID and volume information for each speaker.

property

uid: int | The user ID of the speaker. The uid of the local user is 0.

property

volume: int | The sum of the voice volume and audio-mixing volume of the speaker. The value ranges between 0 (lowest volume) and 255 (highest volume).

property

vad: int | Voice activity status of the local user.

property

channelId: string | The channel ID, which indicates which channel the speaker is in.
